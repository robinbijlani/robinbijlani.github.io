Work

  • Exotics database website (in progress)

    back end development web design

    The Exotics database website is being created for the University of Florida and Florida Fish and Wildlife Conservation Commission to share their data on invasive reptile captures and sightings. Records include measurements, necropsy results, tracking data, etc. The website allows privileged users to view this data and filter it via custom queries. Maps are also generated from each query.

    Notable technologies: PHP, Laravel PHP framework (MVC), Bootstrap

  • EIRAMP tablet app

    mobile app development

    The University of Florida's Everglades Invasive Reptile and Amphibian Monitoring Program (EIRAMP) involves driving along standard routes in the Everglades while identifying all reptiles, amphibians and mammals found. The EIRAMP tablet app facilitates data recording and validates all entered data, greatly reducing human error as well as data entry and proofing time.

    Notable technologies: Cordova/PhoneGap, Angular.js, Bootstrap

  • Triangulate mobile app

    mobile app development

    Triangulate is an Android app built to allow field biologists to accurately estimate the location of tracked animals using triangulation. It uses statistician Russell Lenth's method of maximum-likelihood estimation described in his paper On Finding the Source of a Signal.

    Notable technologies: Cordova/PhoneGap, JavaScript, KendoUI
